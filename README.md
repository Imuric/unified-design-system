### Overview
This repository hosts the interactive dashboard for the Automated Theming Engine and Unified Design System. It serves as a high-fidelity visualization tool for stakeholders to understand the business impact, technical architecture, and design-to-code pipeline described in the design_system_strategy.md blueprint.
### Technical Specifications
- Architecture: Single Page Application (SPA).
- Styling: Tailwind CSS (CDN).
- Visualization: Chart.js (Canvas-based rendering).
- Logic: Vanilla JavaScript for state management and tab-based navigation.
- Responsiveness: Fully adaptive grid layout for mobile, tablet, and desktop viewing.
### Key Features
- Business Impact Analytics: Data visualization comparing manual vs. automated UI refresh timelines.
- Interactive Token Hierarchy: Functional explorer for Primitive, Semantic, and Component token layers.
- Workflow Visualization: Step-by-step breakdown of the Figma-to-Code implementation pipeline.
- Roadmap Tracker: Phased execution plan from foundation to full automation.
### Deployment Instructions
1. Upload index.html to the root of your GitHub repository.
2. Navigate to Settings > Pages.
3. Under Build and deployment, set the source to Deploy from a branch.
4. Select the main branch and the / (root) folder.
5. Save to generate the public URL.
